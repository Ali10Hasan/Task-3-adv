# Little Learners Academy – Landing Page

A responsive landing page for a kindergarten website built using React, TypeScript, and Vite.

## Features

- Responsive design for all screen sizes  
- Mobile-friendly navigation menu  
- Testimonials slider  
- FAQ accordion section  
- Benefits cards  
- Quick navigation links  
- Footer with contact info and social icons  

## Tech Stack

- React  
- TypeScript  
- Vite  
- CSS Modules  
- react-icons
